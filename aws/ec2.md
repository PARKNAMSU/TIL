# EC2

### EC2란?
EC2는 AWS에서 제공하는 Cloud Computing 서비스 이다.

Cloud를 통해 Server, Storage, DataBase 등 컴퓨팅 서비스를 제공, 즉 아마존에서 가상 컴퓨터 한대를 대여하는 것과 같다.

이렇게 대여한 컴퓨터를 Instance 라고 한다.
