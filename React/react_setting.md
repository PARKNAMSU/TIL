# React 개발환경 세팅

##nodeJs가 설치되어 있다고 가정

###폴더 생성 및 이동
터미널에 프로젝트 폴더로 사용할 폴더를 생성한 후 경로를 이동한다.
```
$ mkdir [폴더명]
$ cd [폴더경로]
```

<br>

### react 앱 설치
npx 를 이용하여 react 앱을 설치한다.
````
$ npx create-react-app [프로젝트명]
```

<br>

### 브라우저에서 실행
npm start를 사용하여 브라우저에서 실행해 본다.
```
$ npm start
```
